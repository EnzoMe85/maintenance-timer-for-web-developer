## Description

The "Maintenance Timer for Web Developer" plugin manages and notifies users of the remaining days until the expiration of the website's maintenance. It provides a simple interface within the WordPress administration panel to set the maintenance expiration date and contact details for renewal.

## Features

- **Set Maintenance Expiration Date:** Administrators can set the expiration date for the site's maintenance.
- **Contact Details:** Administrators can specify contact details for renewal.
- **Automatic Notifications:** The plugin will notify administrators based on the set expiration date.

## Installation

1. Download the plugin folder.
2. Upload it to the WordPress plugin directory.
3. Activate the plugin through the 'Plugin' menu in WordPress.

## Usage

1. Once activated, the plugin adds a 'Maintenance Timer' menu in the WordPress administration panel.
2. Go to 'Maintenance Timer' to set the maintenance expiration date and contact details for renewal.
3. The plugin will display alerts in the administration panel based on the expiration date.
4. It also provides a widget on the WordPress dashboard that shows the countdown to the maintenance expiration.
5. ATTENTION: Only those with the Admin role will see and be able to change the settings in 'Maintenance Timer'.
   All other roles will only see the noticeboard and countdown widget.
